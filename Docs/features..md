1. Center Manager (Manager)

Access all system data and functionalities.

Manage pet records: add/delete/edit basic information (name, species, age, weight, brief health status).

Manage services: add/remove basic services (bathing, spa, check-up, vaccination), update prices.

Manage work schedules: assign shifts for staff.

View simple statistical reports: number of services per month, total monthly revenue.

2. Veterinarian

View daily/weekly examination schedules and pet medical records.

Access basic pet information (name, health condition).

Update examination status: record brief diagnosis, start/end time.

Create new medical records or add basic information.
